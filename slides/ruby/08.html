Symbols


  :this_is_a_symbol
Symbols are similar to strings, but they're immutable, and because of that every symbol of the same name refers to the same object in memory. We can test this using the object_id property. 

  # Two strings, even if they have the same value, have different object IDs.
  "a".object_id != "a".object_id

  # Symbols don't though.
  :a.object_id == :a.object_id
